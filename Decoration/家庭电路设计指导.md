# 电路设计

## 整体步骤

1. 确定用电器数量和位置
2. 计算用电器功率大小，初步确定回路分组
3. 根据回路分组，选择导线规格和断路器额定电流
4. 初步设计配电箱
5. 修改设计，应对现场的各种特殊情况
6. 完成

### 回路分组的原则
- 单个回路功率在2000W-5000W之间，超过5000W宜拆分成两个回路，少于2000W则可以和临近回路合并。
- 照明回路单独设置。
- 冰箱和弱电箱等需要长期通电的回路单独设置。
- 卫生间回路宜单独设置，不共享
